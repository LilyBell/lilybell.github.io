Operating Systems

Despite what anyone will tell you the choice of operating system doesn't matter in the grand scheme of things as long as it meets the requirements to run your tools. It is true that PHP is easier to develop with in Ubuntu or macOS but it can be done in Windows as well with a little extra work. My reccomendations for an operating system are Ubuntu 17.10, macOS High Sierra, or Windows 10

IDEs

There are a multitude of viable options for development. PhpStorm is definately my preference but you also have the choice of Atom, Brackets, Sublime, Eclipse, and Netbeans. You can also use vim, nano or emacs if you prefer working from the command line.

Node Packages

There are a ton of reasons to use Yarn over npm. It's faster, it caches everything you use so you can work on new projects offline and it's so much faster but you can still use npm if thats what you're comfortable with.

Database

There are a lot of options here. I work mostly with RDBMSs and Postgres is by and far my favorite but you can use anything you want. You can even use NoSQL if you want!. MySQL, Firebird, Derby, Cassandra, MongoDB, and so on.
